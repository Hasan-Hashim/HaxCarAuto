# Hax Auto Sales

Hax Auto Sales is an e-commerce platform designed for selling cars. This project includes features for browsing vehicles, managing a shopping cart, and performing administrative tasks.

## Features

### User Features:
- **Home Page**: A visually appealing landing page to attract users.
- **Product Listings**: Browse available cars with details like price, name, and images.
- **Product Details**: View detailed descriptions of cars with an option to add them to the shopping cart.
- **Shopping Cart**: Manage selected cars, update quantities, and view the total cost.

### Admin Features:
- **Admin Dashboard**: Manage car inventory with CRUD operations (Create, Read, Update, Delete).
- **Bulk Upload**: Upload multiple car details using JSON or CSV files.
